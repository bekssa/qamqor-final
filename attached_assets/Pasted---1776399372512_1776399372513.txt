Страница поиска и выбора помощников

Нужно реализовать страницу, где пользователь может найти и выбрать помощника для выполнения своей заявки.

1. Основная цель страницы

Пользователь должен иметь возможность:

искать помощников
фильтровать их
просматривать карточки помощников
выбрать помощника
назначить его на одну из своих заявок
отправить заявку напрямую помощнику
2. Структура страницы

На странице должны быть следующие элементы:

1. Поиск

В верхней части страницы находится поисковая строка.

Поиск должен работать по:

имени помощника
категории услуг
ключевым словам из описания

Пример:

Поиск: [Помощь по дому]
2. Фильтр по городу

Рядом с поиском должен быть фильтр города.

Фильтр позволяет показывать только помощников из выбранного города.

Пример:

[Поиск]    [Город: Алматы ▼]
3. Список помощников

Ниже отображается список карточек помощников.

Каждый помощник отображается в виде карточки.

3. Карточка помощника

Каждая карточка должна содержать:

аватарку помощника
имя и фамилию
категорию помощи
рейтинг
краткое описание услуг
кнопку "Выбрать"
Пример структуры карточки
[Аватар]

Имя Фамилия
Категория помощи

⭐ 4.8

Описание:
Помогаю с покупкой продуктов и бытовыми задачами.

[Выбрать]
4. Действие при нажатии "Выбрать"

После нажатия кнопки "Выбрать" система должна проверить:

Есть ли у пользователя активные заявки без исполнителя
5. Если у пользователя есть активные заявки

Открывается модальное окно выбора заявки.

Модалка: выбор заявки

Показывается список заявок пользователя, которые:

активные
не имеют исполнителя

Каждая заявка отображается как карточка:

Заявка #104
Категория: Покупка лекарств
Адрес: Алматы
Дата: 12 мая

[Выбрать эту заявку]

Пользователь выбирает заявку.

После выбора заявки открывается модалка подтверждения помощника.

6. Модалка подтверждения помощника

Модальное окно почти повторяет карточку помощника.

Содержит:

аватар
имя и фамилию
категорию помощи
рейтинг
описание

И две кнопки:

[Отмена]    [Подтвердить]
7. Если у пользователя НЕТ активных заявок

Если у пользователя нет заявок без исполнителя:

пропускается модалка выбора заявки.

Сразу открывается модалка подтверждения помощника.

8. После подтверждения помощника

После нажатия "Подтвердить" открывается страница создания заявки для выбранного помощника.

9. Страница создания заявки

Пользователь должен заполнить:

Поля

1️⃣ Дата выполнения
2️⃣ Время выполнения

3️⃣ Комментарий к заявке

4️⃣ Цена

5️⃣ Адрес выполнения

Выбор адреса

Адрес выбирается по той же логике, как на странице создания заявки на главной странице.

10. Отправка заявки помощнику

После заполнения формы пользователь нажимает:

[Отправить заявку]

После отправки:

заявка отправляется напрямую выбранному помощнику
помощник получает уведомление
11. Уведомления помощнику

Помощник получает:

1. Уведомление
Новая заявка от пользователя
2. Сообщение в чате

В чате появляется новая заявка.

12. Общение

После отправки заявки:

пользователь и помощник могут общаться в чате
обсуждать детали заявки
13. Принятие заявки исполнителем

Исполнитель может:

Принять заявку

или

Отклонить

Если исполнитель принимает заявку:

пользователю приходит уведомление

Пример:

Вашу заявку принял исполнитель
14. Основные состояния заявки

Заявка проходит следующие статусы:

Создана
↓
Отправлена помощнику
↓
Принята исполнителем
↓
Выполняется
↓
Завершена
